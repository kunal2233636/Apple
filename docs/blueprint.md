# **App Name**: BlockWise

## Core Features:

- AI Study Plan Generation: Leverage AI to generate personalized study plans based on the student's JEE syllabus, learning pace, and exam date. Use a tool to determine if/when the study plan will be adaptive.
- Syllabus Breakdown: Automatically break down the JEE syllabus into manageable topics and subtopics for structured learning.
- Progress Tracking: Track the student's progress through the syllabus, identifying strengths and weaknesses. Students will be tracked individually by supabase auth.
- Homepage: Centered card with 'BlockWise' title, subtitle 'AI-Powered Study Management', and a primary blue 'Get Started' button with clean, modern Tailwind design.

## Style Guidelines:

- Primary color: Blue (#2563EB) for a sense of focus and intelligence.
- Background color: Light blue (#EBF4FF), a desaturated version of the primary.
- Accent color: Emerald Green (#10B981) to highlight progress and positive actions, creating contrast against the blue.
- Font: 'Inter' (sans-serif) for both headlines and body text. This font provides a modern, machined, neutral, and objective look suitable for educational content.
- Use clean and simple icons to represent different subjects and topics.
- Modern and clean layout with clear visual hierarchy to guide the user through the study plan.
- Subtle animations to provide feedback on user interactions and progress.
